Note: If you reached this redirect page by clicking the link in the project page, Wikipedia:Double redirects (note the plural as opposed to the singular redirect title above), this is how a double redirect works, which does not take readers to the page they wanted. So the double redirect, which may be linked near the top of this page and titled "Wikipedia:Example of a double redirect", would normally have to be "fixed". Fixing them is easy; see How to fix a double redirect.
