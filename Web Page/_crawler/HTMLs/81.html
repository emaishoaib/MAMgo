Guide

This list is only for the English Wikipedia. Approval is not always needed on other projects. Once you have read this guide, if you want to be registered to use this software, add a request to the bottom of the requests for registration page. Any admin can add any user to the list of enabled users.

Usage

Before using this software, please ensure you are familiar with Wikipedia policies/guidelines, particularly the bot policy and the very simple rules of use:

  • Check every edit before you save it.
  • Don't edit too fast; consider opening a bot account if you are regularly making more than a few edits a minute.
  • Don't do anything controversial with it.
  • Avoid making insignificant minor edits such as only:
    • Adding or removing some white space
    • Moving a stub tag or category name
  • Abide by all Wikipedia guidelines, policies and common practices.
  • Do not edit merely to increase your edit count.

Repeated abuse of these rules will result, without warning, in the software being disabled.

Requesting access

This tool is designed for experienced users to quickly perform specific repetitive tedious tasks. If you don't have a specific need for this then please do not request access for its use. Thank you. (If you're looking for a specific use, typo-fixing is always a good place to start.)

Except in unusual circumstances, users with fewer than 250 non-automated mainspace edits or 500 total mainspace edits will not be added to the list. You can check your non-automated contributions on Tool Labs. If you do not meet this criteria and would like to use AWB for a specific reason, please list that reason when you request access. If your main account meets these criteria, but you want access for a dedicated AWB account, please create the AWB account and then make the request from your main account, replacing the {{subst:REVISIONUSER}} with the username of the requested account.

For security reasons, users who have a history of vandalism or being blocked for similar reasons will probably not be accepted to use the software. The software may also be disabled if the user is blocked for vandalism or disruption.

If requesting access for a bot you will need prior approval. Please include a link to your bot approval page at Wikipedia:Bots/Requests for approval.

Also, to reduce bureaucracy, you will not be notified when/if you are added to the list of enabled users. (Instead, keep this page on your watchlist; when you see your name has been added, you should find you are able to use AutoWikiBrowser). You may add [[Category:Wikipedians who use AutoWikiBrowser|{{PAGENAME}}]] to your user page when you've been added to the list. Users may be removed from this list after a considerable period of inactivity.

If you are an administrator, you are automatically approved. There is no need to add yourself to the user list. However, don't forget AWB Rule 2 ("Consider opening a bot account if you are regularly making more than a few edits a minute.").

If you're wanting to setup a checkpage for other wikis, please see Wikipedia:AutoWikiBrowser/CheckPage format.
